Instructions

The syntax consists of two arguments separated by spaces:

  • The first argument is to tell the program which group of characters you want to include in the output.
  • The second argument is to tell the program the length of the output.

Example 1

  • Input: nl m
  • Output: tjj74s8zw834

Explanation

  • The first argument tells that we want to include numbers (n) and lowercase (l) characters in the output
  • The second argument tells length of the output must be medium (m) = 12 characters

Example 2

  • input: a l
  • output: %LO&}0K0wf2qyOa99EE,5x`]

Explanation

  • The first argument tells that we want to include all group of characters in the output (numbers, lowercase letters, uppercase letters and symbols/punctuations)
  • The second argument tells length of the output must be large (l) = 24 characters

Syntax

1st argument

  • n  or d Output will include numbers
  • l Output will include lowercase letters
  • u Output will include uppercase letters
  • s Output will include symbols/punctuations
  • a Output will include all groups of characters

2th argument

  • s Output length will be 6 (small)
  • l Output length will be 24 (large)
  • x Output length will be 48 (extra large)
  • \d+(one or more digits)  Output length will match the given number (max: 128)

Hotkeys

  • enterSend input
  • ctrl + yCopy output to clipboard
  • ctrl + iFocus on the input box
